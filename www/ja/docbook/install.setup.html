1.2. setup.rbでインストールする

s2container-version.tgz をダウンロードし、setup.rb を実行します。

% tar zxvf s2container-version.tgz
% cd s2container-version
% ruby setup.rb
%

s2container.rb を require します。S2Container.Rubyが使用可能となります。

require 's2container'
