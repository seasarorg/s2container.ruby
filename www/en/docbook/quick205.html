2.5. Constructor Block

Generation of a component and initialization processing can be specified by passing Constructor Block to the s2component method. 

require 's2container'

class Action
  attr_accessor :name
end

s2comp(:class => Action) {
  act = Action.new
  act.name = "foo"
  next act
}

p s2app.get(Action).name    # -> foo
