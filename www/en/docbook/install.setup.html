1.2. Installation with setup.rb

Please download s2container-version.tgz and do "ruby setup.rb". 

% tar zxvf s2container-version.tgz
% cd s2container-version
% ruby setup.rb
%

S2Container.Ruby will be available by requiring "s2container.rb".

require 's2container'
